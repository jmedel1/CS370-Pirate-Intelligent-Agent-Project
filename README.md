# CS 370 Pirate Intelligent Agent Project

## Overview of Project
In this project, I created an intelligent agent using deep Q-learning to solve a pathfinding problem in a maze. The goal of the agent (a pirate) was to learn how to navigate through the maze and reach the treasure while maximizing rewards and avoiding penalties.

I was provided with starter code that included the environment (the maze), helper functions, and parts of the neural network model. The main part I completed was the Q-training algorithm. I implemented how the agent learns through trial and error, updates Q-values, uses exploration vs. exploitation, and improves over time. I also worked with experience replay and a target model to make the training more stable. By the end, the agent was able to consistently find the optimal path to the treasure.

## Problem-Solving Approach
When approaching a problem as a computer scientist, I start by understanding the goal and breaking the problem down into smaller parts. In this project, I focused on understanding how the environment worked, what the agent needed to learn, and how reinforcement learning could be applied.

I then used a step-by-step approach by testing the code, observing results, and making adjustments when needed. Instead of expecting a perfect solution right away, I relied on iteration and improvement, which is an important part of working with algorithms and machine learning.
